# Flora: Your Botanical Companion 🌿

Flora is a premium web application designed for the modern indoor gardener. It combines powerful plant identification technology with personalized care advice and a vibrant community of plant enthusiasts.

## ✨ Key Features

### 1. Instant Plant Recognition
The core of Flora is its AI-powered identification tool. 
- **How it works**: Snap a photo or upload an image of any plant. Our deep-learning model (simulated) analyzes leaf patterns and structure.
- **Result**: You get a 99% accurate identification with confidence scores.

![Plant Identification Result](file:///Users/shahidkhan/.gemini/antigravity/scratch/flora/public/monstera.png)

### 2. Personalized Care Guides
Every identified plant comes with a tailored care schedule.
- **Sunlight**: Precise lighting requirements (e.g., Bright Indirect, Low Light).
- **Watering**: Optimized schedules (e.g., Weekly, Every 2 weeks).
- **Environment**: Ideal temperature and humidity ranges.

### 3. Community Circle
Connect with fellow "plant parents" near you.
- **Swaps**: Trade cuttings and rare species.
- **Advice**: Get help for yellowing leaves or pest issues.
- **Inspiration**: Share your indoor jungle progress.

## 🎨 Design Philosophy
Flora uses a **premium botanical aesthetic**:
- **Palette**: Deep forest greens, sage, and warm creams.
- **Typography**: Modern and elegant "Outfit" and "Inter" fonts.
- **Interactions**: Smooth animations powered by `framer-motion` and glassmorphism UI elements for a state-of-the-art feel.

## 🚀 Getting Started

The application is built with **React**, **Vite**, and **Vanilla CSS**.

### Running the App Locally
1. Ensure you are in the project directory:
   ```bash
   cd flora
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [https://techyshahid.github.io/flora/](https://techyshahid.github.io/flora/) in your browser.
